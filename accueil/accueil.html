Page d'accueil

Bonjour, sur ce site nous allons vous présenter notre blob, qu'on a appelé "Fabrice Besson",
ainsi que les expériences qu'on a réalisées.

Besoin d'aide ?

Pour commencer la découverte du blob, cliquez sur "Présentation du blob".
Ici, vous trouverez des informations sur le blob : comment il se reproduit,
son comportement ainsi que ses facultées étonnantes !

Vous pourrez ensuite cliquez sur "Nos expériences". Cette page vous
présentera les expériences réalisées en S.V.T sur notre blob.
Ces expériences ont été réalisées avec le CNRS dans le cadre du
projet "Élève Ton blob"

Enfin, vous pourrez, si vous le souhaitez, vous rendre sur la page "Crédits".
Là vous trouverez nos noms, ainsi que nos adresses e-mails de contact.







Les sources:

MDN Web Docs, Openclassrooms, Blob's Blog, Le Labo du Blob,
Media VWR, Simply Science, Wikipedia, Le Journal du Dimanche,
Futura Planète, Futura Tech, CNRS
